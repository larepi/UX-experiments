FLOW SUMMARY
Feature Flag + Jira Integration Flow
STEP 1
Navigate to Jira Tab
User opens feature flag configuration and clicks "Jira" tab in sidebar navigation
↓
STEP 2
View Empty State
Empty state displayed with option to add first Jira ticket
↓
STEP 3
Open Add Ticket Modal
User clicks "+ Add Jira Ticket" button, modal dialog opens
↓
STEP 4
Enter Jira Issue Key
User enters Jira issue key (e.g., CBP-12345) in modal input field
↓
STEP 5
Fetch Metadata from Jira API
System retrieves ticket details via Jira API (key, summary, status, URL) - loading state shown
↓
STEP 6
Preview Ticket Details
User reviews fetched ticket metadata before confirming the link
↓
STEP 7
Confirm & Link Ticket
User clicks "Link Ticket" to save the association with the feature flag
↓
STEP 8
View Linked Tickets
Ticket appears in Jira tab as a card showing all metadata with link to Jira
↓
STEP 9
Manage Tickets
User can add more tickets or remove existing ones with confirmation dialog
